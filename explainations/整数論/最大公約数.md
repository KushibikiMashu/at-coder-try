# 最大公約数
gcd(Greatest Common Divisor)

## アルゴリズム
日本語

```
n = (x, yの小さい方)
d = n
loop dがnから1まで
  if dがxとyの約数なら、
    return d
```



## 実装（ライブラリ）
x, yの最大公約数をpythonで求める。mathモジュールを使う

```python
import gcd from math
a = gcd(x, y)
```

## 実装（再帰関数）
```python
def gcd(x, y):
    return gcd(y, x % y) if y else x
```

yが0になったとき、xを返す。そのxが最大公約数
